**Policy Versioning Pattern:**
- `Policy` (container) вҖ” РјРөСӮР°РҙР°РҪРҪСӢРө: slug, name, description
- `PolicyVersion` вҖ” РІРөСҖСҒРёРҫРҪРёСҖРҫРІР°РҪРҪСӢРө РҙР°РҪРҪСӢРө: Р»РёРјРёСӮСӢ, СӮР°Р№РјР°СғСӮСӢ, РұСҺРҙР¶РөСӮСӢ
- `recommended_version_id` вҖ” СғРәР°Р·СӢРІР°РөСӮ РҪР° РІРөСҖСҒРёСҺ РҙР»СҸ РёСҒРҝРҫР»СҢР·РҫРІР°РҪРёСҸ РҝРҫ СғРјРҫР»СҮР°РҪРёСҺ
- РЎСӮР°СӮСғСҒСӢ РІРөСҖСҒРёР№: `draft` (СҮРөСҖРҪРҫРІРёРә), `active` (Р°РәСӮРёРІРҪР°СҸ), `inactive` (РҪРөР°РәСӮРёРІРҪР°СҸ)

## Enum Р·РҪР°СҮРөРҪРёСҸ

### PermissionScope
- `default` вҖ” РіР»РҫРұР°Р»СҢРҪСӢРө РҪР°СҒСӮСҖРҫР№РәРё
- `tenant` вҖ” РҪР°СҒСӮСҖРҫР№РәРё РҙРөРҝР°СҖСӮР°РјРөРҪСӮР°
- `user` вҖ” РёРҪРҙРёРІРёРҙСғР°Р»СҢРҪСӢРө РҪР°СҒСӮСҖРҫР№РәРё

### PermissionValue
- `allowed` вҖ” СҖР°Р·СҖРөСҲРөРҪРҫ
- `denied` вҖ” Р·Р°РҝСҖРөСүРөРҪРҫ
- `undefined` вҖ” РҪР°СҒР»РөРҙСғРөСӮСҒСҸ

### InstanceType
- `local` вҖ” Р»РҫРәР°Р»СҢРҪСӢР№ (РәРҫР»Р»РөРәСҶРёРё)
- `http` вҖ” HTTP API
- `custom` вҖ” РәР°СҒСӮРҫРјРҪСӢР№

### PromptType
- `system` вҖ” СҒРёСҒСӮРөРјРҪСӢР№ РҝСҖРҫРјРҝСӮ
- `baseline` вҖ” baseline РҫРіСҖР°РҪРёСҮРөРҪРёСҸ
- `user` вҖ” РҝРҫР»СҢР·РҫРІР°СӮРөР»СҢСҒРәРёР№

### PromptStatus
- `draft` вҖ” СҮРөСҖРҪРҫРІРёРә
- `active` вҖ” Р°РәСӮРёРІРҪР°СҸ РІРөСҖСҒРёСҸ
- `archived` вҖ” Р°СҖС…РёРІ

### PolicyVersionStatus
- `draft` вҖ” СҮРөСҖРҪРҫРІРёРә, РјРҫР¶РҪРҫ СҖРөРҙР°РәСӮРёСҖРҫРІР°СӮСҢ
- `active` вҖ” Р°РәСӮРёРІРҪР°СҸ РІРөСҖСҒРёСҸ (СӮРҫР»СҢРәРҫ РҫРҙРҪР° РҪР° policy)
- `inactive` вҖ” РҙРөР°РәСӮРёРІРёСҖРҫРІР°РҪР°, РҪРөР»СҢР·СҸ РёСҒРҝРҫР»СҢР·РҫРІР°СӮСҢ

### DocumentStatus
- `pending` вҖ” РҫР¶РёРҙР°РөСӮ
- `processing` вҖ” РҫРұСҖР°РұР°СӮСӢРІР°РөСӮСҒСҸ
- `ready` вҖ” РіРҫСӮРҫРІ
- `failed` вҖ” РҫСҲРёРұРәР°
- `archived` вҖ” Р°СҖС…РёРІ
