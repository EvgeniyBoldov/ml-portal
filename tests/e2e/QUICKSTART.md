# Быстрый старт E2E тестов

## За 3 минуты

```bash
# 1. Запустить сервисы
docker compose up -d

# 2. Применить миграции (дефолтный админ создаётся автоматически)
docker compose exec api alembic upgrade head

# 3. Запустить тесты
cd tests/e2e
./run_tests.sh fast
```

## Что будет протестировано

- ✅ Создание/удаление тенантов и пользователей
- ✅ Создание/удаление чатов
- ✅ Отправка сообщений с идемпотентностью
- ✅ Пагинация и авторизация

**Время:** ~40 секунд

## Полные тесты (включая RAG)

```bash
./run_tests.sh
```

**Время:** ~3-6 минут

## Если что-то пошло не так

```bash
# Проверить, что API работает
curl http://localhost:8000/api/v1/health

# Посмотреть логи
docker-compose logs api

# Перезапустить сервисы
docker-compose restart

# Очистить всё и начать заново
docker-compose down -v
docker-compose up -d
```
