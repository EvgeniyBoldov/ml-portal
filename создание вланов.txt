ИНСТРУКЦИЯ ПО СОЗДАНИЮ VLAN НА РАЗНЫХ ТИПАХ КОММУТАТОРОВ

1. ОБЩИЕ ПРИНЦИПЫ

- VLAN (Virtual LAN) — логическое разделение сети на изолированные домены вещания.
- Каждый VLAN имеет:
  - идентификатор VLAN ID (1–4094, кроме зарезервированных);
  - имя (name), понятное человеку;
  - назначение (user, server, voice, mgmt, guest и т.п.).
- Обязательно ведётся учёт:
  - кто запросил VLAN;
  - дата создания;
  - область применения (какие площадки/этажи/стойки);
  - диапазоны адресов и маршрутизация.

----------------------------------------------------------------------
2. CISCO IOS (L2/L3 КОММУТАТОР)

2.1. ПРОСМОТР СУЩЕСТВУЮЩИХ VLAN

show vlan brief

2.2. СОЗДАНИЕ VLAN И ЗАДАНИЕ ИМЕНИ

conf t
 vlan 10
  name USERS_OFFICE_1
 vlan 20
  name SERVERS_DC_1
end
write memory

2.3. НАЗНАЧЕНИЕ VLAN НА ПОРТ (ACCESS)

conf t
 interface FastEthernet0/1
  switchport mode access
  switchport access vlan 10
  spanning-tree portfast
 end
write memory

2.4. TRUNK-ПОРТ (ПЕРЕДАЧА НЕСКОЛЬКИХ VLAN)

conf t
 interface GigabitEthernet0/24
  switchport trunk encapsulation dot1q
  switchport mode trunk
  switchport trunk allowed vlan 10,20,30
 end
write memory

3. HUAWEI (VRP)

3.1. ПРОСМОТР VLAN

display vlan

3.2. СОЗДАНИЕ VLAN И ЗАДАНИЕ ИМЕНИ

system-view
 vlan 10
  description USERS_OFFICE_1
 vlan 20
  description SERVERS_DC_1
quit
save

3.3. НАЗНАЧЕНИЕ VLAN НА ПОРТ (ACCESS)

system-view
 interface GigabitEthernet0/0/1
  port link-type access
  port default vlan 10
quit
save

3.4. TRUNK-ПОРТ

system-view
 interface GigabitEthernet0/0/24
  port link-type trunk
  port trunk allow-pass vlan 10 20 30
quit
save

----------------------------------------------------------------------
4. JUNIPER (JUNOS, EX/QFX)

4.1. ПРОСМОТР VLAN

show vlans
show interfaces terse | match vlan

4.2. СОЗДАНИЕ VLAN

configure
 set vlans USERS_OFFICE_1 vlan-id 10
 set vlans SERVERS_DC_1 vlan-id 20
commit

4.3. НАЗНАЧЕНИЕ VLAN НА ПОРТ (ACCESS)

configure
 set interfaces ge-0/0/1 unit 0 family ethernet-switching port-mode access
 set interfaces ge-0/0/1 unit 0 family ethernet-switching vlan members USERS_OFFICE_1
commit

4.4. TRUNK-ПОРТ

configure
 set interfaces ge-0/0/24 unit 0 family ethernet-switching port-mode trunk
 set interfaces ge-0/0/24 unit 0 family ethernet-switching vlan members [ USERS_OFFICE_1 SERVERS_DC_1 ]
commit

----------------------------------------------------------------------
5. ОБЯЗАТЕЛЬНЫЕ ШАГИ ПОСЛЕ СОЗДАНИЯ VLAN

- Проверить, что VLAN появился в конфигурации:
  - Cisco: show vlan brief
  - Huawei: display vlan
  - Juniper: show vlans
- Проверить назначение портов:
  - Cisco: show interfaces switchport
  - Huawei: display interface GigabitEthernet0/0/1
  - Juniper: show configuration interfaces ge-0/0/1
- Обновить документацию:
  - добавить VLAN в общий реестр;
  - указать описания портов и назначение;
  - указать маршрутизацию/шлюзы и firewall-политику.
